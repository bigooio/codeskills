# Skills CLI

CodeSkills 命令行工具 - 快速下载和管理编程技能

## 使用方法

```bash
# 一键安装 Skills
npx skills install

# 查看列表
npx skills list

# 更新到最新
npx skills update
```

## 功能

- `install` - 安装/更新 CodeSkills 数据
- `list` - 列出所有已安装的 Skills
- `update` - 更新到最新版本

## 快速开始

```bash
# 在任何目录运行
npx skills install

# 进入下载的目录
cd codeskills-data

# 查看 Skills
npx skills list
```
